# GG Schüler-App

Schüler-App des Gnadenthal-Gymnasium Ingolstadt.
Ermöglicht Zugriff auf den aktuellen Vertretungsplan, die Nachrichten der Schulleitung sowie ein schwarzes Brett der SMV.
